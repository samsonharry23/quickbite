# 🍕 QuickBite

A fast, modern **pizza-ordering app** — browse the menu, build your cart, place an order, and track it in real time. No login required; just enter your name and start ordering.

## ✨ Features

- 🍕 **Live menu** loaded from a REST API
- 🛒 **Cart** with quantity controls and live price totals
- 📝 **Place an order** with name, phone, address, and optional priority
- 📍 **Geolocation** to auto-fill your delivery address
- 🚀 **Order tracking** — look up any order by ID and mark it priority
- 💛 Clean, responsive UI built with Tailwind CSS

## 🛠️ Tech Stack

- **React** (Vite)
- **Redux Toolkit** — global cart & user state
- **React Router** (data loaders & actions) — routing + data fetching
- **Tailwind CSS** — styling
- REST API for menu & orders

## 🚀 Getting Started

```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173`.
